# デザインルール

## 原則

画面は面接前の緊張を下げることを最優先にします。情報量を絞り、進行中の練習、残り時間、次に押すボタンがすぐ分かる構造にします。

## トーン

- 背景は明るいニュートラルを基調に、要所だけ深いグリーンとコーラルを使う。
- 紫、濃紺、ベージュ一色の印象に寄せない。
- ボタンとカードの角丸は8px以下にする。
- 説明文は機能説明ではなく、利用者に向けた短い言葉にする。

## レイアウト

- 初期画面はランディングページではなく、すぐ練習を始められるダッシュボードにする。
- 主要な練習画面はカードに閉じ込めず、画面幅を使って集中できる構成にする。
- メッシュアニメーションは面接官の存在感を表す中心要素として扱う。
- 履歴、課金、振り返りは練習の邪魔にならないサイド領域に置く。

## アクセシビリティ

- 主要操作はキーボードだけで完了できる。
- 音声入力が使えない環境でもテキスト入力で練習できる。
- 色だけで状態を示さず、ラベルや数値も併記する。
