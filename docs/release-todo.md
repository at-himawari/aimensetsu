# リリースTodo

AI面接コーチを初回リリースするまでのチェックリストです。まずはMVPの本番公開に必要な項目を優先し、音声入力や文書解析の高度化はリリース後に段階的に育てます。

- [ ] MVPの機能範囲を確定する。
  - 完了条件: 初回リリースに含める機能と含めない機能がREADMEまたは仕様メモに明記されている。
- [ ] 本番/ステージングの環境分離方針を決める。
  - 完了条件: フロント、API、DB、Stripe、Cognito、Azure OpenAIの環境名と接続先が整理されている。
- [ ] 利用規約、プライバシーポリシー、特商法表記、返金ポリシーを用意する。
  - 完了条件: フッターまたは設定画面から各文書へ遷移できる。
- [ ] 本番で扱う個人情報と保持期間を決める。
  - 完了条件: メール、氏名、電話番号、職務経歴書、会話履歴、決済情報の保存方針が文書化されている。
